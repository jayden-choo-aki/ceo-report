▸ 재무 진단 — 무엇이 둔화시키고 있는가
  • 매출 가속 종료 — 4년 가속(+23%·+14%·+5.6%·+0.85%) 후 정체. FY2025 매출 3,041억은 사상 최대지만 모멘텀 소진.
  • 마진 동시 압박 — FY2023→2025 매출원가율 36.3%→38.8% (+2.5pt), 판관비율 49.0%→50.8% (+1.8pt). 손익 양면 동시 악화로 OP% 14.6%→10.4% 후퇴.
  • 지분법손실 누적 — FY2024·2025 지분법손실 합계 60억. 종속·관계기업(상해법인·뉴욕법인·베트남법인) 적자가 별도 손익에 직접 반영되는 구조.
  • 연결 영업손실 시그널 — FY2025 연결 영업이익 287억(별도 316억보다 29억 낮음) = INDONG FN VINA 베트남 신규 편입분이 영업단 손실. 자산화 단계(투자) → 회수 단계(매출) 전환이 늦어지면 마진 추가 압박.
  • 배당 vs 재투자 균형 — FY2024 배당총액 200억(순이익 236억 대비 85%) → FY2025 200억 유지. 배당성향 높이는 가운데 마진 후퇴는 자본 누적 속도를 떨어뜨림.

02 재무 진단 · Financial Diagnosis KPI·매출·이익·재무구조·수익성·비용·효율성·현금흐름. 21년 시계열로 변화량과 추세를 점검.

수익성 · 마진 트렌드

매출총이익률·영업이익률·순이익률 추이. 패션업 글로벌 평균 GP% 56.88%(NYU Stern), OP% 우수기업 10%+.

해석: FY2022~2023 OP% 14~15%로 역대 최고 구간 (글로벌 패션 우수기업 10%+ 통과). FY2024~2025 OP% 11~12%로 하락 — 매출원가율 36→38% 악화 + 판관비율 49→51% 동반 상승이 누적된 결과.
시사점: 1pt 마진 회복 = 약 30억 영업이익. 매출 성장보다 마진 복원의 ROI 우위. 매출원가율과 판관비율 양 트랙 동시 개선이 필수.

비용 구조 분해 · FY2025 판관비 1,546억

판관비 항목별 비중. 판매수수료가 65%로 압도적 — 외부 채널 의존도가 마진 직격. 채널 협상력·D2C 전환이 판관비율 50.8% 개선의 핵심 레버.

FY2025 판관비 Top 6 항목

해석: 판매수수료 1,011억(65.4%) → 매출의 33%. 급여 199억(12.9%)은 고정비. 임차료 95억은 매장·물류 운영 비용. 시사점: 1pt 판관비율 개선 = 약 30억 영업이익. 판매수수료(채널) 또는 임차료(점포) 둘 중 어디를 줄이느냐의 의사결정.

비용 구조 종합 (매출 대비 비중)

해석: 매출 3,041억 = 매출원가 1,179억(38.8%) + 판관비 1,546억(50.8%) + 영업이익 316억(10.4%). FY2023 정점 OP% 14.6% 대비 -4.2pt 후퇴. 시사점: 매출원가율(+2.5pt) + 판관비율(+1.8pt) 양측 동시 개선 필요. 한 쪽만으로는 OP% 13% 복귀 불가.

효율성 지표 · 자본·자산 회전

21년간 ROE·ROA·회전율 추이. 무차입(이자보상배율 N/A) 외에 자본·자산이 얼마나 효율적으로 매출/이익을 만들어내는지 평가. 패션업 우수기업 ROE 15~20% 벤치마크.

ROE · ROA (수익성 효율)

해석: FY2005 ROE 29%(자본 작음·고레버리지) → FY2013 33% 정점 → FY2025 14.4%. 시사점: 자기자본 누적(분모↑)이 순이익 성장(분자)을 앞서며 자본 효율성 정체. 패션 우수기업 15~20% 대비 미달 구간 진입.

자산·자기자본 회전율 (활동성)

해석: 자산회전율 FY2013 1.51회 → FY2025 1.68회 (자산 활용 양호). 자기자본회전율 FY2005 14.7회(고레버리지) → FY2025 2.15회 (자본 누적으로 하락). 시사점: 자산은 효율적, 자본은 활용 여력 존재 — 배당·재투자 정책의 트레이드오프 신호.

▸ CCC(현금전환주기) 산출 한계 — 감사보고서에서 재고자산·매출채권·매입채무가 유동자산/유동부채에 합산 표시되어 분리 불가. DSO/DIO/DPO는 별도 시스템 데이터 필요. 본 페이지의 "운전자본 부담률 = (유동자산-현금)/매출"은 proxy 지표.

03 자산·자본 구조 · Structural Detail 자산구성·운전자본·자본배분·종속기업·연도별 감사보고서. 구조적·운영적 지표와 원자료.

자산 구성 분석

유동자산(재고·매출채권·현금)과 비유동자산(토지·건물·시설·상표권·종속기업 투자) 추이.

유동 vs 비유동자산

해석: 비유동자산 step-up 시점 — FY2012(이천 토지 26.7억·건설중자산 46.3억), FY2014(상표권 27.4억), FY2024~25(지분법 적용투자주식 + 베트남 자산). 시사점: 비유동자산 비중이 자산총계의 22~25% 박스권으로 운전자본 중심 패션업 특성 유지.

비유동자산 비중 (자산 내)

해석: 비유동비중 20~25% — 패션업 평균과 일치. 시사점: 자산 증가의 주축은 여전히 재고·매출채권. 운전자본 효율 관리가 자산 효율성의 핵심 레버.

운전자본 효율 분석

재고자산·매출채권 회전율과 운전자본 부담률 추이. 패션업 핵심 효율 지표.

매출 대비 운전자본 부담률

해석: (재고+매출채권)/매출 비율. 시사점: 비율이 높아질수록 매출 1원당 묶이는 운전자본 증가 — 마진 회복 시급도와 직결.

재고자산 회전율 (회/년)

해석: 매출원가 ÷ 평균재고. 패션업 일반 4~8회. 시사점: 회전율 하락은 재고 적체·세일 의존도 증가 신호. 마크다운 비용을 통한 마진 압박 가능성.

자본 배분 분석

순이익 → 배당 vs 사내유보 vs 차입금 상환의 자본 배분 패턴.

배당총액 & 배당성향

해석: FY2010 첫 배당(3.9억) → FY2024~25 200억으로 51배 확대. 배당성향 FY2024 85%, FY2025 98%. 시사점: 사내유보 속도 둔화 → 자기자본 증가 정체 가능성. 마진 회복 없이 고배당 지속은 자본 누적 마이너스 신호.

유동부채 (차입금 포함) 추이

해석: FY2013 정점(910억)에서 FY2025 382억까지 -58%. 단기차입금 FY2021 무차입 진입. 시사점: 재무 안정성 확립 단계 완료. 차입 여력 충분 — 성장 투자 의사결정 가능 구간.

종속·관계기업 손익 영향

감사보고서 강조사항 — 종속·관계기업의 손실이 별도 손익에 미치는 영향 (지분법손실 기준).

영업외수익 vs 비용 (지분법손실 포함)

별도 vs 연결 영업이익 격차 (종속기업 영향)

해석: 연결발행 3개년(FY2016·17·25) 모두 연결 영업이익 < 별도 → 종속기업이 매년 영업단 손실. FY2017 -25억, FY2025 -29억으로 격차 확대. 시사점: 종속기업 정상화 시 연결 OP +29억 자동 회복 = OP% 약 1pt 상승 효과.

종속·관계기업 누적 손익 패턴: ① 인동(상해)복장유한공사 — FY2007 설립, FY2016~2017 연결 시점 누적 순손실 -40억대(매출 84~115억, 순손실 -15.7억·-24.7억). ② INDONG FN NEW YORK INC — 외감 미대상 규모지만 지분법 적용 대상. ③ INDONG FN VINA — FY2025 신규 연결, 자산 122억·영업손실 -29억·순손실 -30억.
시사점: 해외 종속기업이 별도 손익을 압박하는 구조. 연결 손익 개선 = 종속기업 정상화. 베트남은 신규 자산투자 회수 단계 진입 전, 상해는 누적 손실로 추가 자본투입 vs 정리 의사결정 시점.
