## 2. 섹션별 세부 — 데이터 항목 & 표현 규칙

### ① 의사결정 안건 TOP 3 (NEW)

| 컬럼 | 내용 | 예시 |
|------|------|------|
| 안건 | 한 줄 요약 | "SISTINA FW 입고량 +20% 추가 발주 여부" |
| 배경 수치 | 변화량 1~2개 | "현재 sell-through 68% (목표 55% 초과)" |
| 옵션 | 2~3개 | A: 추가 발주 / B: 보충 없이 마감 / C: 일부 SKU만 |
| 기한 | 결정 마지노선 | "5/30까지 (베트남 공장 라인 확보)" |
| 영향 추정 | 매출·재고 영향 | "+18~25억 매출 / +12억 재고 리스크" |

→ **CEO가 매주 받는 3가지 결정 요청**. 결정 후 다음 주 보고서에 "결과·반영" 표기.

### ② 변곡점 트래커 (성장 둔화 대응) (NEW)

가설 3개와 회복 신호 3개를 **고정 트래커**로 운영:

| 둔화 원인 가설 | 측정 지표 | 이번 주 값 | 추세 |
|---------------|----------|----------|------|
| 가설 1: SHESMISS 핵심 고객 이탈 | 30대 후반 재구매율 | xx% | ↓ / → / ↑ |
| 가설 2: 백화점 채널 노후화 | 백화점 동매점 매출 YoY | xx% | — |
| 가설 3: 신상품 적중률 하락 | New % of Sales (입고 6주차) | xx% | — |

| 회복 신호 | 측정 지표 | 이번 주 값 | 임계치 |
|----------|----------|----------|--------|
| 자사몰 D2C 비중 | 전체 매출 대비 | xx% | 25% |
| SISTINA 매출 비중 | 전체 매출 대비 | xx% | 15% |
| 베트남 공장 내재화율 | 자체 생산 비율 | xx% | 40% |

→ 가설·임계치는 CEO 검토 후 분기마다 갱신.

### ③ 재고·공급망 헬스 (가중치 ★★★★)

- **Sell-through**: 브랜드×주차×카테고리 매트릭스 (heatmap)
- **재고 일수 (Days on Hand)**: 브랜드별, 시즌 잔여 일수와 함께
- **마크다운율**: 신상품 → 정상가 → 세일 → 클리어런스 단계별 매출 비중 (waterfall)
- **베트남 공장 가동률·납기 준수율**: 라인별
- **내재화율**: 자체 생산 / 외주 OEM 비율 (전체·브랜드별)
- **시즌 입고 진척**: 계획 대비 입고 완료 %

### ④ 브랜드별 성과 (가중치 ★★★★)

| 브랜드 | 주간 매출 | YoY | 마진 | Sell-through | 베스트/워스트 SKU |
|--------|----------|-----|------|--------------|------------------|
| SHESMISS | | | | | |
| LIST | | | | | |
| **SISTINA** | | | | | ★ |

**SISTINA 별도 트래커** (2021 런칭 후 누적):
- 누적 매출·점유율·점포 수
- 객층(MZ 타깃) 도달도: 20~30대 매출 비중
- 회사 전체 매출 기여도 추세

### ⑤ 자사몰(IDFmall) (가중치 ★★★★)

- **GMV, 전환율(CVR), AOV, 객수**: 주간, 4주 이동평균
- **신규회원 / 재구매율 / 멤버십 등급별 매출**
- **D2C 비중** (전체 매출 대비) — 분기 임계치 도달 여부
- **Funnel**: 세션 → 장바구니 → 결제 (단계별 drop-off)
- **모바일 vs 데스크탑 / 앱 vs 웹**

### ⑥ 채널 성과 (가중치 ★★★, 월간)

- 백화점 / 아울렛 / 자사몰 / 해외 / B2B 매출·비중·YoY
- 점포 단위: 평당 매출, 동매점 매출 YoY
- 신규/폐점

### ⑦ 재무 스냅샷 (가중치 ★★★, 월간, 변화량 위주)

- 매출·영업이익·마진 (예산 대비 %, YoY %)
- 매출원가율·판관비율 (트렌드)
- 운전자본·재고자산·현금성 자산 (베트남 공장 투자 후 변동)
- **표현 규칙**: 절대값은 보조, **±% 변화량을 시각적으로 강조**

### ⑧ 부록

- SKU·매장 단위 디테일
- ESG·지속가능성 진척 (분기)
- 인적자원: 인당 매출, 이직률, 핵심 인재 (분기)
- 경쟁사 동향: 한섬·신세계인터내셔날·LF (분기 IR 시점)
- 거시: 소비심리, 환율, 면화·유가 (월간)

| ① 의사결정 안건 | ● | ● | ● | ● |
| ② 변곡점 트래커 | ● (값) | ● (해석) | ● (가설 갱신) | — |
| ③ 재고·공급망 | ● | ● (요약) | ● (시즌 회고) | — |
| ④ 브랜드별 성과 | ● | ● | ● | ● |
| ⑤ 자사몰 | ● | ● | ● | ● |

| ⑦ 재무 스냅샷 | — | ● | ● | ● |

## 4. 표현 원칙 (페르소나 반영)

1. **수치 → 메커니즘 → 결정안건** 3단 구조. "매출 -5%" 다음 줄에 "원인 가설", 그 다음 줄에 "필요 결정".
2. **Variance 우선**: "예산 ±%", "YoY ±%"를 절대값 앞에 둠.
3. **변곡점 가시화**: 추세 화살표(↑→↓), 임계치 도달 여부 색상.
4. **장기 결정의 트래커화**: 베트남 공장·자사몰·SISTINA는 매번 별도 박스로 진척 표기.
5. **외부 비교**: 분기 보고에 한섬·신세계인터내셔날·LF 동기 수치 한 줄 인용.
6. **선언적 톤 지양**: "글로벌 강화 진행 중" 같은 표현 금지 → "베트남 공장 가동률 78% → 82% (+4pt)" 식.
