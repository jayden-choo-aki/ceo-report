## 2. ★ Headline Strip — Inditex Tell-Twice 핵심 (NEW)

### 양식

```
─────────────────────────────────────────────────────────────────────
WEEK 21 · 2026-05-18~24 · 보고: 2026-05-25 (다음: 2026-06-01)
─────────────────────────────────────────────────────────────────────
매출 W21    │  OP W21    │  GM         │  D2C       │  재고일수  
58.3억      │  6.8억     │  56.2%      │  18.4%     │  62일      
(예산 ‑4%)  │  (예산 +2%)│  (+30 bps)  │  (+2.1pt)  │  (‑3일)    
(YoY +3%)   │  (YoY +12%)│  (YoY ‑5bp) │  (YoY +5pt)│  (YoY ‑8일)
─────────────────────────────────────────────────────────────────────
Sell-through│ 베트남가동 │ ₩/USD       │
68.3%       │  82%       │  1,372      │
(목표 65%↑) │  (계획 80%)│  (월초 1,358│
            │            │   → +1.0%)  │
─────────────────────────────────────────────────────────────────────
```

→ **CEO가 시간이 없으면 이 한 줄만 봐도 결론 도달**. 이후 섹션은 같은 숫자의 **메커니즘**을 풀어주는 역할.

### 규칙

- 8개 숫자만, 더 늘리지 않음
- 모든 숫자에 **Variance 2종** 동반 (예산 대비 / YoY)
- 임계치 미달 시 색상 (Red/Yellow/Green)
- 1줄 형태 — 표가 아닌 띠(strip)

## 3. ★ 6-12 차트 (Amazon WBR 표준) — NEW

### 양식

```
지표명: Sell-through Rate (SHESMISS)
┌─────────────────────────────┬─────────────────────────────────┐
│   좌: 최근 6주 (주간)        │   우: 최근 12개월 (월간)         │
│                              │                                 │
│      ●─●─●─●─●─●            │      ●─●─●─●─●─●─●─●─●─●─●─●  │
│   ─ ─ ─ ─ ─ ─ 목표선         │   ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ 목표선   │
│                              │                                 │
│   W16  17  18  19  20  21    │   '25/06 ─────────  '26/05      │
└─────────────────────────────┴─────────────────────────────────┘
이상치 (Red/Yellow/Green) │ 가설 (Why) │ 액션 (What next)
```

### 적용 대상 (Weekly 5개 필수)

| # | 차트명 | Input/Output/Financial | 비고 |
|---|--------|------------------------|------|
| 1 | Sell-through Rate (브랜드별) | Output | 가장 중요 |
| 2 | 재고 일수 (Days on Hand) | Output | 베트남 공장 입고 변동 추적 |
| 3 | 베트남 공장 가동률 + 납기 준수율 | Input | 본인 결정의 ROI |
| 4 | 자사몰 CVR + GMV | Output | 2019 결정의 ROI |
| 5 | 마크다운율 (전체 매출 대비) | Financial | 마진 압박 선행지표 |

### 메트릭 순서 캐논

본문 흐름은 **Input → Output → Financial** 순서로 고정:
- Input: 가동률·입고·신상품 출시·MD 액션
- Output: Sell-through·재고일수·전환율·객수
- Financial: 매출·마진·마크다운·재고평가

→ "왜 그렇게 됐는지(Input)" 먼저, "결과(Output)" 다음, "돈으로 얼마(Financial)" 마지막. CEO가 메커니즘을 따라가게 됨.

| Op 마진 / GM | ##% | ##% | ##% | CVR ##% |
| Sell-through / KPI 2 | ##% | ##% | ##% | AOV ##원 |
| Comment (1줄) | "..." | "..." | "..." | "..." |

### 별표(★) 처리

- **SISTINA** = 2021 본인 결정의 검증 → 누적 진척(매장수·매출비중) 1줄 추가

→ Fast Retailing이 UNIQLO Japan/International/GU/Global Brands를 한 슬라이드에 묶듯, 인동에프엔 4축을 한 페이지로.

## 5. ⑦ 재무 + 한국 특화 KPI 5종 (UPGRADE)

기존 v2 재무 스냅샷에 **한국 패션업 특수성 5종 추가**:

### 5-1. 백화점 채널 수수료 분석

| 점포 등급 | 점포 수 | 매출 비중 | 평균 수수료율 | 가중 수수료액 |
|----------|--------|----------|--------------|--------------|
| 신세계 강남 등 프리미엄 | | | ~38% | |
| 현대·롯데 본점급 | | | ~30% | |
| 지방 거점 | | | ~24% | |
| **전체 가중평균** | | | **23.7% 업계평균 대비 ±**| |

→ "백화점 매출 비중이 줄어도 수수료 가중평균이 오르면 마진 악화" 같은 메커니즘 감지.

### 5-2. 특약매입 vs 직매입 비중

- 특약매입 = 재고 리스크 백화점 부담 / 직매입 = 인동에프엔 부담
- 비중 변화는 백화점 매입 정책 변동의 선행지표

### 5-3. S/S vs F/W 시즌 곡선

- 한국 여성복은 F/W가 약 50% (소비자 객단가·아우터 가격 효과)
- 누적 진척 곡선을 작년 동기 곡선과 overlay
- 시즌 후반 sell-through 부진 시 **마크다운 위험 사전 경고**

### 5-4. 아울렛/행사매장 매출 기여

- 아울렛 비중 ↑ = 정상가 sell-through 부진의 **역지표**
- 분기별 추세를 별도 라인으로

### 5-5. 환율·면화·베트남 가동률 민감도

| 변수 | 1% 변동 시 영향 (월간) | 현재 값 | YoY |
|------|----------------------|--------|-----|
| ₩/USD | 원가 약 ±__억 | 1,372 | +%% |
| ₩/VND | 베트남 인건비 ±__억 | __ | __ |
| 면화 선물 | 원자재 ±__억 | __ ¢/lb | __ |
| 베트남 가동률 1pt | 영업이익 ±__억 | 82% | __ |

→ 외부 변수가 본인 결정(베트남 공장)에 미치는 영향을 정량화.
