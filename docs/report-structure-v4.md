## 3. 데이터 가용성 매핑 (전체 KPI 라벨)

각 KPI 옆 라벨 강제:

- **[Y]** = 현재 ERP/POS/IDFmall로 즉시 집계
- **[Partial]** = 일부 가능, 정의 합의 또는 수기 보완 필요
- **[N]** = 신규 시스템 또는 외부 소스 필요
- **[?]** = 미확인, 데이터팀 점검 필요
- **⏸ Defer** = 데이터 수집 SOP 우선

**즉시 가능한 KPI (Y)** 대부분
**데이터 SOP 우선 항목** (⏸): 백화점 수수료율 가중평균, 디자이너 이직률 (HRIS 정의), Scope 1+2 (측정 체계)
**외부 소스 필요** (N): EU DPP 대응 진척, EMV/NPS 측정 도구
