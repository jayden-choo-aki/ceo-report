## 1. 패션업 CEO가 봐야 하는 핵심 정보 (5개 도메인)

### A. 재무·수익성

| 지표 | 의미 / 벤치마크 |
|------|----------------|
| 매출 / 영업이익 / 순이익 | 절대값 + YoY 성장률, 예산 대비 달성률 |
| 매출총이익률 (Gross Margin) | 패션 우수 기업 55~65%, NYU Stern 평균 56.88% |
| 순이익률 | 우수 8~15%, 업계 평균 3.85% |
| 매출원가율 | 상승 = 세일 의존도 증가 신호 (한국 패션 공통 악화 요인) |
| 판관비율 | 한국 패션 대형사 약 50% (신세계인터내셔날 사례) |
| 현금성 자산 / Net Cash | Inditex FY25 €11.0B 보유 — 재무 안정성 척도 |

### B. 재고·공급망 (패션의 생존 지표)

| 지표 | 의미 / 벤치마크 |
|------|----------------|
| Sell-Through Rate (판매소진율) | (초기 입고 + 보충) 대비 판매 비율. MD/Buyer 핵심 KPI. 시즌 중 매주 추적 |
| 재고자산회전율 | 일반 패션 4~8회/년, fast-fashion 더 높음 |
| GMROI (재고투자수익률) | 재고 1원당 매출총이익 |
| 재고 일수 (Days on Hand) | 시즌 종료 시점 잔여 재고 |
| 마크다운율 (할인 판매 비중) | 한국 패션 마진 압박의 주범 |
| 시즌 진척률 / 입고 지연 | OEM·ODM 의존 시 필수 |

### C. 채널·고객 (옴니채널)

| 지표 | 비고 |
|------|------|
| 채널 mix | 백화점 / 아울렛 / 자사몰(DFmall) / B2B / 해외 비중 |
| 자사몰 전환율 (CVR) | 패션 e-com 목표 25% (18개월 내), 일반 e-com 평균 ~2% 대비 높게 잡음 |
| AOV (객단가) / 객수 | 주간 추적 |
| 매장 단위 매출 / 평당 매출 | 오프라인 효율 |
| CAC vs LTV | 마케팅 ROI |
| 재구매율 / 멤버십 전환율 | LTV의 선행 지표 |
| 세션·바운스·Cart Abandon | 디지털 funnel 진단 |

### D. 브랜드·제품

| 지표 | 비고 |
|------|------|
| 브랜드별 매출 비중 | 인동에프엔: SHESMISS / LIST / SISTINA 3분할 |
| 카테고리 mix | Top / Bottom / Dress / OPS 등 |
| 신상품 매출 기여도 (New % of Sales) | 트렌드 적응력 |
| Best/Worst SKU | 재발주·소진 판단 |
| 컬렉션 라이프사이클 진척 | 입고 → 정상가 → 세일 → 클리어런스 |

### E. 전략·외부환경 (분기·연간)

- **ESG / 지속가능성** — 인동에프엔 모토에 포함된 영역
- **글로벌 공급망 / 해외 진출** — 자체 보도상 4,500억원 목표의 핵심 축
- **경쟁사 동향** — 한섬·신세계인터내셔날·LF 분기 IR 비교
- **거시 지표** — 소비심리, 환율, 면화·유가, 트렌드 키워드
- **인적자원** — 인당 매출, 이직률, 핵심 인재 확보

## 3. 샘플 / 참고 리포트

### 글로벌 (구조 참고)

- **Inditex (Zara 모회사) FY2025 Results** — 매출, GP/GP%, EBITDA/EBIT, Net Cash, 채널·지역별 성장, 신사업(Zara Try-on AI fitting 등)을 한 문서에 담는 모범 구조
- **LVMH** — 세그먼트(Fashion & Leather, Wines, Perfumes 등)별 Revenue/Op Income 한 페이지 요약
- 시장 규모: LVMH 약 $90B(글로벌 1위), Inditex 4위 (TJX·Nike 사이)

### 국내 (직접 비교군)

- **한섬 (현대백화점그룹)** — IRGO 페이지에 사업보고서·분기보고서 정리. 자체(TIME/MINE) + 수입(Tommy 등) 구조 — 인동에프엔과 비교 적합
- **신세계인터내셔날** — 사업보고서 PDF 공개. **판관비 50% 이슈** — 비용 구조 벤치마크
- **한세실업** — OEM/ODM(GAP·H&M·Target 등 바이어) 관점 사업보고서 — 공급망 관점 참고
- **DART 전자공시** (`dart.fss.or.kr`) — 정공식 사업보고서·감사보고서 원본. **인동에프엔 외감법인이므로 여기서 조회 가능**

### 템플릿

- Modeliks, ClickUp, Cascade — Fashion/Apparel KPI 대시보드 템플릿 (한국 중견 패션 맞춤은 아니지만 항목 매핑용)
